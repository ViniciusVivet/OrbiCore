# OrbiCore

Plataforma visual de gestao para pequenos negocios. O nucleo operacional da empresa.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- Recharts
- Supabase (auth + postgres)
- ExcelJS

## Desenvolvimento

```bash
npm install
npm run dev
```

## Deploy

Hospedado na Vercel.
